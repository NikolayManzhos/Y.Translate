# Translator
Test assigment for Yandex Mobilization project.
## Features
* Handling configuration changes.
* All requests are automaticaly cached.
* Unit tests.
* Simple and clean interface.
* Different layout for landscape and portrait.
## Used libraries
* RxJava 2
* Dagger 2
* Butter Knife
* Rerofit 2
* Realm
* Leak Canary
